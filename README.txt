REQUISITI

Per utilizzare questo programma è necessario avere installato sul proprio sistema Python 3.

UTILIZZO

Da un qualsiasi terminale o prompt dei comandi, recarsi nella directory root del progetto(dove è posizionato il file requirements.txt) ed eseguire il seguente comando:

pip install -r requirements.txt

In questo modo si installerano tutte le dipendenze necessarie.

Per far partire il programma, eseguire il file Python main.py utilizzando il comando

python main.py

Questo farà partire l'ottimizzazione utilizzando il modello già addestrato (maggiori dettagli nella relazione).
Verranno generati diversi file, tra cui quello contenente i valori di input ottimi, che verrà salvato in model/x_star.txt.
Verrà inoltre generato un grafico per l'andamento della funzione totale f(x).
Al termine, verrà automaticamente avviato il programma fornito dal docente, per verificare il valore trovato.
Verrà infine generato e mostrato un grafico dei punti trovati.

Per maggiori dettagli, vedere la relazione allegata.

Il file trovato x.txt è comunque già presente in questa cartella.
